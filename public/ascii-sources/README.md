# ASCII source images

Source images for the `/ascii-lab/` and `/about/` text-raster experiments. The site converts these source SVGs into animated typographic raster graphics in the browser.

- `compass-rose-black.svg` — Wikimedia Commons file [`Compass_Rose-Black.svg`](https://commons.wikimedia.org/wiki/File:Compass_Rose-Black.svg); used for the Strategy card.
- `icon-gears2.svg` — Wikimedia Commons file [`Icon-gears2.svg`](https://commons.wikimedia.org/wiki/File:Icon-gears2.svg); used for the Operations card.
- `recycling-symbol.svg` — Wikimedia Commons file [`Recycling_symbol.svg`](https://commons.wikimedia.org/wiki/File:Recycling_symbol.svg); used as the iteration-cycle source image for Product & Program Leadership.
- `brain-outline-lateral.svg` — Wikimedia Commons file [`Brain-outline-lateral.svg`](https://commons.wikimedia.org/wiki/File:Brain-outline-lateral.svg); used for the AI & Automation card.
- `social-network.svg` — Wikimedia Commons file [`Social-network.svg`](https://commons.wikimedia.org/wiki/File:Social-network.svg); used for the Stakeholder Networks card.
- `compass-source.svg` — local fallback SVG for earlier compass experiments.
- `stakeholder-network-source.svg` — local fallback SVG for earlier stakeholder network experiments.
